## 二、redis 数据类型

- string （类比 java 中 String）
- hash （类比 java 中 HashMap）
- list （类比 java 中 LinkedList）
- set （类比 java 中 Hash）
- sorted_set （类比 java 中 TreeSet）

redis 自身是一个 map，其中所有的数据都是采用 key:value 的形式
数据类型是指存储的数据类型，也就是 value 的类型，key 永远是字符串
